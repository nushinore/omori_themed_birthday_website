1. <p></p> =  paragraph text
2. <pre></pre> = pre-formatted text
3. <head></head> = information about the website (like website header)
4. <title></title> = used for putting the title in the head tag
5. <body></body> = body of the html page
6. <h1> , <h2>, <h3>, <h4> = header text (1>4 in size) (all have closing tags)
7. <br> = breakline, move to the next line
8. <hr> = horizontol row; draws a horizontal line on the screen
9. <a></a> = anchor tag (attributes: href(link), title(on hover), target(if you wanna open in another tab))
10. <img> = adds an image  (attributes: src (link to the image), alt(alternate text when image doesnt load), height, width) 
11. <audio></audio> = adding audio to the website (attributes: controls (to add controls to play the audio), autoplay (autoplay the audio)  )
12. <source></source> (under audio and video and image tag) =  (attributes: src(source link of the audio), )
13. <video> = attributes: src , height, width
